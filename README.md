# Bozorbob E-commerce Core

A full-stack e-commerce application focused on storefront delivery, authentication, and catalog management for a multilingual business use case.

**Live demo:** https://bozorbob-uz-original.vercel.app

## What This Project Shows
- End-to-end product delivery for a commerce domain
- Authentication and account-oriented flows
- SSR-oriented storefront thinking
- Flexible product data modeling with MongoDB

## Why This Stack / Tooling
- **Next.js** was chosen for route structure and storefront rendering that benefits from SEO.
- **MongoDB + Mongoose** support a flexible catalog and evolving product shapes.
- **JWT + bcrypt** handle practical authentication needs for a custom commerce app.
- **Axios and cookie-based session helpers** keep client/server communication straightforward.
- **Swiper / Slick** support merchandising-heavy UI patterns common in storefront experiences.

## Stack
- Next.js 12
- React 17
- MongoDB / Mongoose
- JWT
- bcrypt
- Axios
- Swiper / React Slick

## Key Features
- Product browsing and storefront pages
- Auth-oriented customer flows
- Rich media and carousel-driven merchandising
- Multilingual commerce foundation
